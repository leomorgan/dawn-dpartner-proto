1. LOW EXPLAINED VARIANCE:
   - PC1+PC2 explains only 54.7% of total variance
   - This means most variation is in higher dimensions (PC3-PC64)
   - 2D projection loses critical differentiating information

2. FEATURE SATURATION:
   - 13/64 features have near-identical values across all sites
   - These features contribute no discriminative power

3. NORMALIZATION ISSUES:
   - Average variance per dimension: 0.0425
   - ⚠️  Very low variance suggests features are poorly scaled
   - Most features clustered near same values (0.5-0.8 range)

4. SMALL DATASET:
   - Only 11 samples in 64D space
   - PCA needs ~10x more samples than dimensions for reliable projection
   - Recommended minimum: ~640 samples for 64D

RECOMMENDATIONS:

A. Increase dataset size to 50+ captures for better PCA
B. Use t-SNE instead of PCA (better for small datasets)
C. Select top-K most variant features (e.g., top 10-20 features)
D. Review normalization ranges - ensure features span 0-1 effectively
E. Consider UMAP for non-linear dimensionality reduction
